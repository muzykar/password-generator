usage: command_line.py [-h] [--words WORDS] [--separator SEPARATOR]
                       [--prefix PREFIX] [--chain] [--length LENGTH]
                       [--sentence] [--language {en}]

Generate random password

optional arguments:
  -h, --help            show this help message and exit
  --words WORDS         Number of words in password. Range of words: 2-19
  --separator SEPARATOR
                        Type of words separator
  --prefix PREFIX       First letter of words. Number of words is ignore
  --chain               Chain of words
  --length LENGTH       Length of password. Range of password length: 12-127
  --sentence            Adjective + noun + verb. Number of words is ignore
  --language {en}       Select language
